# Get Versions

This runs the Github cli command to get all the tigerbeetle releases and calculates the latest version a release can update to.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.34. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
